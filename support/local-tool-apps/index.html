Local Tool Apps Support

This support page applies to local utility apps published by this developer when the App Store product page links to this page.

Common Test Path

  1. Open the app.
  2. Use the main input, worksheet, calculator, planner, validator, or report tab.
  3. Enter sample values or use the built-in example content.
  4. Run the calculation, validation, planning, or report action.
  5. Copy, save, share, or review the output if the app includes that workflow.
  6. Open Settings or About to review privacy and app information.

Privacy and Account Notes

The covered apps do not require login, account registration, analytics, advertising, tracking, or cloud upload. Core workflows are designed to run locally on device.

Payment Notes

When an app is sold as a paid download, its core features are available after purchase. The covered apps do not use external payment links, license keys, or manual unlock flows for digital features.

Troubleshooting

  • If output looks unexpected, check that the input units match the labels shown in the app.
  • If copied output is missing, run the calculation or report action again before copying.
  • If saved local history is not needed, use the app's clear or delete controls where available.
  • If the app was recently updated, restart the app after installation to refresh local state.
